* Yufan Survives JS

** Getting Started
*** Set up [[https://webpack.github.io/][Webpack]]
1. =npm init=
2. =npm i webpack node-libs-browser --save-dev=
3. set up -webpack.config.js-
4. set up entry point -build/index.html-
